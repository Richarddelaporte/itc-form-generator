# ITC Form Generator

**Generate Inspection, Testing & Commissioning forms from Sequence of Operation documents.**

## Quick Start

1. **Double-click** `ITC_Form_Generator_Web.exe` to start the application
2. Your browser will automatically open to `http://localhost:8080`
3. Upload your SOO document and optional points list
4. Download your generated ITC forms (HTML, CSV, or Excel)

## Features

- **📄 Document Parsing** - Supports PDF, Markdown, and text SOO documents
- **📊 Points Integration** - Import BMS points lists (CSV/TSV/Excel)
- **🤖 AI Enhancement** - Optional AI-powered form generation for better results
- **📚 Template Library** - Pre-built templates for RSB and ATS equipment
- **📥 Multiple Exports** - Download as HTML, CSV, or Excel

## System Requirements

- Windows 10 or later
- No Python installation required (standalone executable)
- Modern web browser (Chrome, Edge, Firefox)

## Sample Files

- `sample_soo.md` - Example Sequence of Operations document
- `sample_points_list.csv` - Example BMS points list

## Troubleshooting

**App won't start?**
- Ensure no other application is using port 8080
- Try running as Administrator
- Check Windows Firewall settings

**Browser doesn't open?**
- Manually navigate to `http://localhost:8080`

**Forms not generating?**
- Ensure your SOO document is in a supported format (PDF, MD, TXT)
- Check that the file is not empty or corrupted

============================================================
            ITC FORM GENERATOR - Quick Start Guide
============================================================

HOW TO RUN:
-----------
1. Double-click "ITC_Form_Generator_Web.exe"
2. Your browser will open automatically to http://localhost:8080
3. Upload your SOO document and generate forms!

SUPPORTED FILES:
----------------
- SOO Documents: PDF, Markdown (.md), Text (.txt)
- Points Lists: CSV, TSV, Excel (.xlsx)

FEATURES:
---------
- Generate ITC forms from Sequence of Operations documents
- Import BMS points lists for enhanced verification items
- AI-powered form generation (optional)
- Pre-built templates for RSB and ATS equipment
- Export to HTML, CSV, or Excel

TROUBLESHOOTING:
----------------
If the app doesn't start:
  - Make sure port 8080 is not in use by another app
  - Try running as Administrator

If browser doesn't open:
  - Manually go to http://localhost:8080

SAMPLE FILES INCLUDED:
----------------------
- sample_soo.md         - Example SOO document
- sample_points_list.csv - Example points list
